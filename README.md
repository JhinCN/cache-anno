自动缓存方法数据注解

基于AOP注解实现的自动缓存注解，支持Redis和本地缓存，用最简单的方式来读写缓存数据

本仓库包含以下内容：

1. @Cache注解一个
2. 关于@Cache对应的自动缓存实现
